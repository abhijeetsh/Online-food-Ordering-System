# Online-food-Ordering-System
you need to download zipped file
install xampp server to run php and mysql locally
extract your zipped folder into your xampp directory c/xammp/htdocs/orderonline etc.
go to localhost/phpmyadmin/
create database and name as fos_db
go to import option and import the sql from database folder
now go to the browser and type localhost/orderonline/
